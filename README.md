# angular-advanced

Practice project: angular-advanced.

## Overview
This repository serves as a practice ground for advanced Angular concepts and best practices.

## Features
- Advanced routing techniques
- State management with NgRx
- Lazy loading modules

## Installation
Clone the repository and install the dependencies:

```bash
npm install
```

## Usage
Run the application:

```bash
ng serve
```
